# 01_产品边界说明

> 关联文档：《无限画布AI视频生成平台_产品开发需求文档.docx》
> 版本：v0.1（草稿）

## 1. 一句话定义

面向日历类周边产品（新年台历、新年挂历）的**批量图生视频生产平台**：在无限画布上以节点化分镜模板组织生产流程，替换素材即可批量产出 25～30 秒的产品展示视频。

## 2. 用户范围

- 主要用户：日历类产品的运营/设计人员，非专业视频剪辑人员
- 非目标用户：本期不面向专业剪辑师提供精细化时间轴调色/关键帧动画能力

## 3. 功能边界（In Scope）

| 模块 | 说明 |
|---|---|
| 无限画布 | 节点摆放、连线、缩放平移、批量框选 |
| 节点体系 | 素材节点、AI生成节点、分镜节点、合成节点 |
| 分镜模板 | 结构化、可复用的镜头序列模板，标准提示词字段 |
| 素材管理 | 按角色绑定素材、素材需求清单前置规划 |
| 批量生产 | 模板克隆、批量任务队列、失败重跑 |
| 生成质量策略 | 约束运镜、可选版面/文字保护叠加方案 |
| 导出 | 按平台规格导出成片 |
| 产品线 | 新年台历、新年挂历两条线，共用节点体系，差异集中在运镜展示与场景镜头 |

## 4. 明确不做（Out of Scope）

- **不做**逐帧精细剪辑/关键帧曲线调整能力，分镜节点参数只开放结构化预设，不开放时间轴级自由编辑
- **不做**多角度实拍图的 3D 重建/自由视角合成（详见 03、09 中的技术边界说明）
- **不做**复杂字幕特效引擎、无智能配音/文案生成（字幕若需要，MVP 阶段只支持固定样式叠加）
- **不做**电商店铺自动上架/自动发布到抖音等平台的集成，导出后由人工分发
- **不做**多人协作权限体系、评论/审批流（MVP 阶段单用户操作，团队协作能力列入 V2 待评估）
- **不做**移动端适配，本期为桌面 Web 端

## 5. MVP 范围内的产品线取舍

MVP 阶段优先跑通**单一产品线**（建议挂历，因其"场景镜头+运镜展示"更具展示性，便于验证核心链路），台历作为第二条产品线在 MVP 之后接入，具体阶段划分见《05_MVP开发阶段拆分.md》。

## 6. 核心术语边界（与 PRD 一致）

| 术语 | 边界说明 |
|---|---|
| 节点 Node | 画布最小功能单元，四类固定，不支持用户自定义新节点类型（MVP阶段） |
| 分镜模板 Template | 仅保存结构与固定字段，不保存具体绑定的图片 |
| 素材角色 Asset Role | 每个产品线预定义固定角色集合（如主图/细节特写1/细节特写2/场景图），MVP 阶段不支持用户自定义新角色 |
| AI生成节点 | 定位为素材清单缺口的**兜底手段**，不作为默认供图路径 |
