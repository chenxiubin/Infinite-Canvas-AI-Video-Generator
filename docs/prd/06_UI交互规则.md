# 06_UI交互规则

## 1. 画布基础操作

| 操作 | 手势/触发 | 说明 |
|---|---|---|
| 缩放 | 滚轮 / 触控板双指 | 无级缩放，无最大层级限制（性能允许范围内） |
| 平移 | 空格+拖拽 / 中键拖拽 | 与常见设计工具（Figma）保持一致，降低学习成本 |
| 框选 | 鼠标左键拖拽空白区域 | 可批量选中节点或产品链，用于批量克隆 |
| 多选 | Shift/Ctrl+点击 | 追加选中 |
| 节点拖拽 | 直接拖拽节点卡片 | 仅调整画布内位置，不改变节点间连线的逻辑顺序 |

## 2. 节点卡片视觉状态规则

### 2.1 类型色（不随生成状态变化）
| 节点类型 | 颜色 | 备注 |
|---|---|---|
| 素材节点 | 灰色 | |
| AI生成节点 | 粉色 | 候选锁定后自动转为素材节点视觉（灰色），生成节点本身的配置保留在模板层，不常驻画布 |
| 分镜节点-可变 | 紫色 | 随产品变化的镜头（主图/细节/运镜/场景） |
| 分镜节点-固定 | 珊瑚色 | 品牌固定镜头（如尾帧LOGO） |
| 合成节点 | 灰色 | 系统自动执行，无需人工配置 |

### 2.2 生成状态边框（叠加在类型色之上）
| 状态 | 边框样式 |
|---|---|
| 未生成 pending | 灰色细边框 |
| 生成中 generating | 蓝色跑马灯动画边框 |
| 成功 success | 绿色边框 |
| 失败 failed | 红色边框，仅影响该节点 |

### 2.3 清单联动标记（V1 起）
- 角色状态为 `pending`（待补充）：节点显示灰色虚线边框 + "待补充：{角色名}" 文案
- 角色状态为 `fulfilled`：恢复正常类型色

## 3. 节点编辑抽屉规则

- 触发方式：单击节点，右侧展开抽屉；画布主体保持可见，不使用遮挡式模态弹窗
- 抽屉内字段**全部为下拉预设**（景别、运镜方式、光线氛围、转场方式、运镜幅度），**不提供自由文本输入框**，避免批量场景下风格不一致
- 时长字段为滑块，范围 3～5 秒，拖动时对应节点卡片宽度实时同步变化（时长与卡片宽度成正比，便于用户直观感知节奏分布）
- 主体变量位区域：展示当前绑定素材缩略图 + 来源标签（"素材"或"AI生成"）+ 更换图片按钮
- 关闭抽屉方式：点击画布空白处 / 点击其他节点切换 / 右上角关闭按钮

## 4. 连线交互规则

| 触发 | 行为 |
|---|---|
| 单击连线 | 弹出小气泡，选择转场类型（预设选项：淡入/交叉溶解/直切） |
| 悬停连线中点 | 浮现"+"按钮，点击在此处插入新节点 |
| 插入新节点后 | 自动重新编号（node_key 递增），总时长实时刷新 |
| 总时长超出 30 秒 | 顶部时长数字变红，提示"还可插入 X 秒"或提示需删减节点/压缩时长 |

## 5. 批量绑定交互规则

1. 用户在左侧素材库框选多组产品素材（每组对应一个 SKU 的完整角色素材）
2. 拖拽到已配置好的模板链上
3. 拖拽过程中展示实时预览动画："1条链正在裂变成N条"，明确告知批量范围
4. 松手后触发批量克隆，画布上纵向排列生成 N 条独立链
5. 批量克隆操作支持撤销（Ctrl+Z），撤销后已创建的实例及其生成任务同步取消

## 6. 折叠/展开规则

- **默认状态**：分镜节点链默认按"素材角色分组"展开显示（因产品导向的模板节点数量本身不多，6～7个节点，无需像原月份版本那样默认折叠）
- 若后续模板节点数量增加（如细节特写扩展到3～4个），单条链横向超出画布可视宽度时，允许用户手动折叠为分组卡片，展开状态优先保证信息可见性
- 该默认值为设计假设，需在原型可用性测试中验证，非最终定论（对应01文档中的待确认事项）

## 7. 批量任务看板交互规则

- 汇总进度条格式：`{总数}条链，已完成{n}，生成中{n}，失败{n}`
- 点击失败项 → 跳转回画布并定位、高亮到具体失败节点
- 失败节点支持"单独重跑"按钮，重跑不影响同批次其他产品链的状态
