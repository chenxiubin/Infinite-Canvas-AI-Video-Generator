# 21_MVP-3_演示版本总说明

> 版本：v0.3-mvp-demo
> 日期：2026-07-03

## 1. 版本定位

基于无限画布可视化交互的 AI 电商视频批量生产平台 MVP 演示版。

## 2. 当前支持产品线

- desk_calendar：新年台历
- wall_calendar：新年挂历

## 3. 当前已跑通能力

### 产品素材包
- 创建产品
- 登记素材（file_url mock）
- role_key 自动识别（main/detail1/detail2/scene/motion/brand）
- 手动确认 role
- checklist（missing_required_roles / unconfirmed_required_roles 拆分）
- motion fallback（motion → scene → main）

### 视频模板
- 台历默认模板（26s, 6 shots）
- 挂历默认模板（26s, 6 shots）
- 模板查询

### 批量链路
- video batch 创建
- video instance 生成
- 6 个 video instance nodes
- 素材自动绑定 + motion fallback
- batch_tasks 复用为 video batch 主表

### Mock 生成
- batch generate（同步 mock）
- node generate
- failed node retry（attempt_no, retry_of_job_id）
- success node skip
- generation jobs 留痕
- node/instance/batch 状态联动
- TESTING=true force 注入，TESTING=false 403 拦截

### 审核导出
- merge preview（draft_preview_url）
- review_records
- node approve/reject
- instance batch approve/reject
- mock export（final_video_url）
- archived batch 阻断 merge/review/export
- retry/regenerate 后 delivery state 重置

### 前端生产工作台
- 创建 Demo 产品素材包
- 一键运行完整 Demo
- 状态总览（6 卡片）
- Demo 步骤日志
- 工作台/画布视图切换
- Reset 页面状态
- Error 信息可见

### 无限画布
- 6 节点横向可视化
- status / review_status 颜色
- 节点详情面板（prompt/video_url/cover_url）
- approve/reject from detail panel
- zoom in/out/reset
- empty state
- canvas 与 form 状态同步

### 测试
- backend: 124 tests
- MVP-2 E2E: 14 tests
- MVP-3 workbench E2E: 2 tests
- MVP-3 canvas E2E: 5 tests
- MVP-3 demo E2E: 3 tests
- Total E2E: 24 tests

## 4. 当前不做范围

- 不接真实图生视频模型（可灵/即梦/Runway/Flow/RunningHub）
- 不做真实 ffmpeg 合成
- 不做真实视频文件下载
- 不做真实文件上传
- 不做自动视觉质检
- 不做复杂时间轴剪辑
- 不做自由节点连线编辑
- 不做多用户权限
- 不做团队审核流
- 不做投放平台上架

## 5. 本地启动步骤

### 后端

```bash
cd backend
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

如 8000 被占用（WinError 10013），换端口并同步修改 frontend/vite.config.ts proxy：

```bash
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8010
```

### 前端

```bash
cd frontend
npm install
npm run dev
```

访问：`http://127.0.0.1:5173`

## 6. 演示流程

1. 打开 `http://127.0.0.1:5173`
2. 工具栏点击「生产工作台」
3. 点击「一键运行 Demo」
4. 等待状态总览显示：
   - 产品 ready
   - 模板 selected
   - batch completed
   - nodes 6/6 success
   - review approved
   - export success
5. 查看 final_video_url（/mock-exports/...）
6. 切换到「画布视图」
7. 查看 6 个分镜节点（S01→S06）
8. 点击节点查看详情（shot_key/status/video_url/prompt）
9. 查看 canvas-draft-preview-url 和 canvas-final-video-url

## 8. 当前风险清单

- Mock 图生视频不代表真实模型效果
- 产品图案/日期/文字保护未接入真实模型验证
- Mock preview/export 只是 URL，非真实视频文件
- 真实 ffmpeg 合成未实现
- 真实文件上传未实现
- batch_tasks 复用为 video batch 主表，后续真实调度需扩展
- 一键 Demo 适合演示，不等同真实批量生产
- 画布为可视化/半交互，不是模板设计器
- 无权限系统和多人审核流
- 无成本统计和模型失败原因细分
- 无自动视觉质检
