# MVP-4 前端节点式工作流交互定稿

> 文档版本: Sprint 10A
> 状态: 定稿，待审批
> 最后更新: 2026-07-06

---

## 一、产品定位

1. 平台不是传统表格式批量生成工具。
2. 平台不是完全自由的 ComfyUI。
3. 平台是"固定电商视频工作流模板 + 节点式画布 + 半自动 AI 生成工作台"。
4. 用户核心操作是:
   上传素材 → 替换参考图 → 分镜生成 → 视频预览 → 审核 → 总合成 / 下载。

---

## 二、三栏职责

### 左侧: 悬浮素材 / 流程栏

- 默认收缩成长条图标栏。
- 鼠标 hover 展开。
- 包含: 产品线切换(台历/挂历)、素材库入口、图片素材 tab、视频素材 tab、分镜快速定位、素材需求清单。
- 左侧不是主要操作区，只做资源与导航。

### 中间: 节点画布(主操作区)

- 固定工作流结构。
- 显示参考图节点、分镜控制节点、视频预览节点、总合成节点。
- 支持拖拽、粘贴、连线、生成、预览。

### 右侧: 属性 Inspector

- 选中节点后右侧滑出。
- 只显示当前选中节点的详细参数。
- 选中分镜节点时显示: prompt 参数、高级提示词、参考图绑定、生成按钮。
- 选中视频预览节点时显示: 视频播放、审核/驳回/重做。
- 右侧不是全局控制台，而是节点详情区。

---

## 三、顶部状态栏

- 保留顶部状态栏。
- 不再使用大卡片，做成轻量悬浮状态条。
- 显示: 素材、模板、批次、生成、审核、导出。
- 状态条与画布节点状态联动。
- 不占用大量画布高度。

---

## 四、固定工作流布局

1. 台历/挂历是两个固定工作流布局。
2. 切换台历/挂历时，工作流布局切换，当前连线和节点内容重新开始，素材库保留。
3. 固定工作流中核心节点: 参考图节点、分镜控制节点、视频预览节点、总合成节点。
4. 固定节点默认连线已存在，用户只需替换图片即可使用。
5. 用户可以手动改线。
6. 默认节点布局必须规整，避免图片比例变化后互相遮挡。
7. 画布可以缩放，参考图节点允许按比例自适应高度。

## 五、节点类型定义

### 5.1 ReferenceImageNode / 参考图节点

| 属性 | 说明 |
|------|------|
| 职责 | 放置首帧、尾帧、普通参考图 |
| 空状态 | 默认 3:4 竖屏，显示提示文字"首帧参考 / 拖入参考图" |
| 图片自适应 | 根据宽高比自适应，支持 3:4、9:16 等竖屏比例 |
| 连线 | 可连到一个或多个分镜节点 |
| 删除 | 支持，不删除素材库图片 |
| 清空 | 支持，保留节点和连线，不删除素材库图片 |
| 替换 | 新图片继承原连线 |

### 5.2 ShotControlNode / 分镜控制节点

| 属性 | 说明 |
|------|------|
| 职责 | 固定工作流主骨架 |
| 删除 | 不允许 |
| 新增 | 不允许新增真正分镜 |
| 显示 | 分镜名称、状态、参考图缩略图、生成按钮 |
| Inspector | 点击后右侧显示详细参数 |
| 生成 | 节点上有快速生成按钮，Inspector 中也保留，共用同一 handler |
| 缩略图规则 | 1 张图显示"首帧"，2 张图显示"首帧+尾帧"，>2 张显示"参考图1/参考图2/+N" |
| 排序 | 用户可在分镜节点内拖动参考图排序，排序持久化，影响 payload 和 prompt |

### 5.3 VideoPreviewNode / 固定视频结果节点

| 属性 | 说明 |
|------|------|
| 职责 | 每个分镜默认固定存在，表示该分镜当前生效的视频结果 |
| 删除 | 不允许 |
| 参与审核 | ✅ 是 approved-only gate 判断依据 |
| 参与总合成 | ✅ 总合成只读此节点的当前视频版本 |
| 未生成 | 显示"等待生成" |
| 已生成 | 封面 + 播放按钮，允许节点内播放 |
| 播放规则 | 同一时间只允许 1 个视频播放，点击另一个时自动暂停上一个 |
| 替换 | 新生成视频直接替换当前槽位，历史版本保留在视频素材库 |
| 从素材库替换 | 从视频素材库拖入历史版本到此节点上，直接替换当前结果，review_status 重置为 pending |
| 清空 | 支持，清空后分镜回到待生成 |
| 状态 | approved 显示"已通过，可合成"；rejected 显示"已驳回"+原因；rejected/failed 可显示"重新生成" |
| 审核 | 不在节点内做 approve/reject，审核仍在 Inspector |
| 状态源 | 不作为业务状态源 |

### 5.5 VideoCompareNode / 自由视频对比节点

| 属性 | 说明 |
|------|------|
| 职责 | 用于对比播放历史视频，不参与审核和总合成 |
| 创建方式 | 右键画布空白处"新建自由视频对比节点"，或从视频素材库拖入历史视频创建 |
| 删除 | ✅ 可以删除 |
| 参与审核 | ❌ 不参与 |
| 参与总合成 | ❌ 不默认参与 |
| 播放规则 | 同一时间只允许 1 个视频播放 |
| 升级为固定版本 | 拖动此节点到对应分镜的固定 VideoPreviewNode 上，替换当前版本，review_status 重置为 pending |
| 状态源 | 不作为业务状态源 |

### 5.4 MergeNode / 总合成节点

| 属性 | 说明 |
|------|------|
| 职责 | 总合成预览+下载 |
| 删除 | 不允许 |
| 显示条件 | 所有分镜审核通过后才显示 |
| 默认状态 | 暂停 |
| 功能 | 播放最终合成预览、"一键下载当前所有视频分镜" |
| 下载规则 | 只下载每个分镜当前使用中的视频版本，不下载历史视频，按分镜顺序命名 |
| 素材库 | 总合成视频不进入视频素材库 |

## 六、参考图交互规则

1. 三种放图方式:
   - **拖到画布空白处**: 进入素材库，并在当前位置生成参考图节点。
   - **拖到固定参考图节点**: 进入素材库，替换该节点图片。
   - **hover + Ctrl+V**: 进入素材库，替换该节点图片。
2. Ctrl+V 没有 hover 节点时: 图片进入素材库，在当前画布位置生成参考图节点。
3. 所有进入画布的图片都必须进入全局素材库。
4. 节点显示的是素材库图片引用，不是只存在节点内。
5. 替换图片不弹确认。
6. 替换图片不提示"旧视频基于旧参考图"。
7. 替换图片后，连接到它的所有分镜缩略图必须立刻同步。
8. 删除参考图节点: 节点消失→连线删除→分镜失去该参考→不删素材库图片。
9. 清空参考图图片: 节点保留→连线保留→图片清空→不删素材库图片。
10. 一键清空参考图: 清空固定参考图节点里的图片→节点占位保留→素材库不清空。

---

## 七、连线规则

1. 默认工作流连线预置完成。
2. 用户可手动改线。
3. 一个参考图节点可以连接多个分镜。
4. 新建参考图节点默认孤立，连线后才归属分镜。
5. 连线必须持久化，刷新页面后不能丢。
6. 连线关系影响分镜缩略图。
7. 连线关系影响生成 payload。
8. 连线关系影响 prompt 中参考图编号。
9. 手动改线后不弹"是否重新生成"提示。

---

## 八、右键菜单规则

| 位置 | 选项 |
|------|------|
| 画布空白处 | 新建参考图节点、新建自由视频对比节点 |
| 参考图节点 | 清空图片、删除节点 |
| 固定视频结果节点 | 清空当前视频 |
| 自由视频对比节点 | 清空视频、删除节点 |
| 分镜节点 | 查看属性(不显示删除) |
| 总合成节点 | 查看总合成详情(不显示删除) |

第一版右键菜单只保持必要选项，不塞全局操作。

## 九、素材库规则

1. 素材库是全局素材库。
2. 台历/挂历切换时素材库保留，不按产品线拆分。
3. 素材库分两个 tab: 图片素材、视频素材。

### 图片素材 tab
- 全部图片统一展示。
- 支持搜索和简单筛选。
- 不按首帧/尾帧/参考图分类。
- 不标记已使用/未使用。

### 视频素材 tab
- 保存每次分镜生成成功的视频。
- 保留所有历史版本。
- 按分镜分组展示。
- 显示: 分镜编号、分镜名称、生成时间/次数、审核状态、当前使用中标记。

---

## 十、历史视频版本规则

1. 每次分镜生成成功，自动进入视频素材库。
2. 每个分镜保留所有历史视频版本。
3. 新生成视频直接覆盖当前视频预览槽位，被覆盖版本仍保留在视频素材库。
4. 历史视频可从素材库拖到画布空白处，生成自由 VideoCompareNode 用于对比观看。
5. 历史视频只能替换同一个分镜的当前结果，不允许跨分镜替换。
6. 跨分镜拖错时提示: "该视频属于 S01 主图-正面，不能替换 S03 细节特写。"
7. 历史视频拖回对应分镜的固定视频结果节点时: 直接替换当前结果→review_status 重置为 pending→用户需重新审核。
8. 当前使用中的历史版本标记"当前使用中"。
9. 历史视频不能直接拖到总合成节点。总合成只读每个分镜当前使用中的视频版本。
10. 删除普通历史版本: 直接删除。
11. 删除当前使用中版本: 弹确认"该视频正在作为当前分镜结果使用，删除后当前分镜将回到待生成状态，是否继续？"，确认后移除，不做回收站。

---

## 十一、播放规则

1. 视频预览节点默认暂停。
2. 总合成节点默认暂停。
3. 点击某个播放时，其他视频自动暂停。
4. 第一版允许节点内播放，不允许多个视频同时播放。

---

## 十二、删除/清空/替换规则汇总

| 对象 | 可删除 | 可清空 | 影响素材库 | 影响连线 | 影响分镜状态 | 需确认 |
|------|--------|--------|-----------|---------|------------|--------|
| 参考图节点图片 | - | ✅ | 不删 | 保留 | 分镜失去参考 | 否 |
| 参考图节点本身 | ✅ | - | 不删 | 删除连线 | 分镜失去参考 | 否 |
| 分镜节点 | ❌ | - | - | - | - | - |
| 固定视频结果节点内容 | - | ✅ | 不删 | - | 分镜回到待生成 | 否 |
| 固定视频结果节点本身 | ❌ | - | - | - | - | - |
| 自由视频对比节点 | ✅ | - | - | - | 不影响分镜 | 否 |
| 总合成节点 | ❌ | - | - | - | - | - |
| 历史视频版本 | ✅ | - | 删除 | - | 当前使用中版本影响分镜 | 仅当前版本 |
| 替换参考图 | - | - | 不删 | 继承 | 不自动清空旧视频 | 否 |

## 十三、持久化规则

1. 参考图节点与分镜节点连线必须持久化。
2. 参考图排序必须持久化。
3. 自由新增节点位置必须持久化。
4. 固定工作流默认节点位置由模板决定。
5. 刷新页面后连线、排序、自由节点位置、当前使用中视频版本不能丢。
6. 台历/挂历切换时: 工作流布局整体切换，当前产品线连接和节点状态按对应工作流加载，素材库保留。

## 十七、已确认规则(10A 定稿)

### 17.1 台历/挂历切换规则

1. 素材库全局保留，切换产品线不清空。
2. 切换产品线时，目标产品线工作流按默认布局重新开始。
3. 当前参考图绑定、生成视频、审核状态、连线状态不跨产品线继承。
4. 后续如果做项目级保存，可作为独立项目状态处理。

### 17.2 视频历史版本数量

1. 第一版不设硬上限。
2. UI 默认展示每个分镜最近 20 个历史版本。
3. 后续可增加"展开更多 / 清理历史版本"功能。

### 17.3 一键下载格式

1. 总合成节点提供 zip 下载。
2. 只下载当前每个分镜正在使用中的视频版本，不下载历史版本。
3. 文件按分镜顺序命名:
   - `01_S01_main.mp4`
   - `02_S02_detail1.mp4`
   - `03_S03_detail2.mp4`
   - `04_S04_motion.mp4`
   - `05_S05_scene.mp4`
   - `06_S06_brand.mp4`

### 17.4 默认参考图节点数量

1. 普通分镜默认 1 个参考图节点。
2. S04 运镜展示默认 2 个参考图节点。
3. 其它额外参考图节点通过右键新建或用户手动连线添加。
