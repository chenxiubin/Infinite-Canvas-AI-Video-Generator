# 分镜规划与提示词属性面板设计方案

> 关联文档：07_素材角色与分镜模板规则.md（本方案是对该文档分镜结构的更新迭代）
> 版本：v0.2（草案，待实际测试验证后定稿）

## 一、设计原则

本方案的核心前提是**单帧驱动生成**（不采用首尾帧插值、不采用多角度3D重建），原因是首帧图只提供了一个固定角度的信息，凡是要求模型"脑补"画面之外内容的操作，都会带来结构失真风险。围绕这条原则，本方案在运镜方式、运动幅度等字段上做了针对性限制：

- **只保留不需要模型脑补画面外信息的运镜方式**：静止、推进、拉远、轻微平移。**环绕从选项中移除**——环绕的本质是展示首帧图未拍到的侧面/背面，与单帧驱动的稳定性诉求根本冲突，不通过"提醒用户注意幅度"来处理，直接从下拉选项层面移除。
- **运动幅度只保留轻微/中等两档**，"较大"从选项中移除，理由与上一条一致：从源头上掐掉高风险选项，不依赖用户自我克制。
- 高级模式提供**可选的系统安全后缀**，用一句固定描述强化"不脑补画面外结构"这一诉求，效果需要通过实际测试验证（见第六节待测试事项）。

## 二、分镜结构总览（6镜头，无品牌固定节点）

参考电商产品广告的惯例结构：开场吸睛 → 细节证明品质 → 动态/尺寸展示 → 使用场景建立信任 → 收尾呼应。品牌LOGO/水印**不再作为独立镜头**，如需要通过合成阶段的图层叠加实现，不占用AI生成镜头位置（当前阶段暂不需要水印叠加）。

| 镜头 | role_key | 建议时长 | 内容 | 备注 |
|---|---|---|---|---|
| S01 开场主图 | `main` | 3～4s | 产品完整正面展示，建立第一印象 | |
| S02 细节特写1 | `detail_1` | 3～4s | 材质/纸张质感，证明做工 | |
| S03 细节特写2 | `detail_2` | 3～4s | 结构/工艺细节，证明品质 | |
| S04 动态展示 | `motion` | 4～5s | 主方案：翻页/开合定格瞬间；备用方案：尺寸参考同框 | 待测试项，详见第五节 |
| S05 场景镜头 | `scene` | 4～5s | 生活场景陈列，建立使用画面感 | |
| S06 收尾呼应 | `closing` | 3～4s | 回到产品整体展示，与S01首尾呼应 | 运镜方向与S01相反，形成"进→出"叙事闭环 |

**台历/挂历产品线差异**：沿用07号文档已定义的差异（S03结构细节内容、S04动作类型、S05场景类型按产品线分别定义），本方案不改变这部分，只更新镜头总体结构与属性字段设计。

## 三、全局字段约束（适用于全部6个镜头）

| 字段 | 可选项 | 说明 |
|---|---|---|
| 景别 | 特写 / 中景 / 远景 | |
| 运镜方式 | 静止 / 推进 / 拉远 / 轻微平移 | **不含环绕**，全镜头统一约束 |
| 光线氛围 | 清透高饱和(默认) / 暖光节日氛围 / 侧光质感 / 自然生活光 / 冷调简约 | 全部镜头默认值统一为"清透高饱和"，其余选项保留供后续按镜头调整 |
| 运动幅度 | 轻微(默认) / 中等 | **不含较大**，全镜头统一约束，具体轻微/中等由用户按实际效果测试调整，不做镜头级别的差异化默认 |
| 背景虚化程度 | 无虚化 / 轻度虚化(默认) / 强虚化 | 新增字段，见下方定义 |
| 安全边距 | 0%～20%（默认8%） | 新增字段，**仅当运镜方式为"推进"或"拉远"时显示**，其余运镜方式下隐藏此字段 |

### 背景虚化程度 —— 字段定义

控制产品与背景的清晰度对比。虚化程度越高，背景越模糊、产品越突出，但要求首帧图本身有一定景深层次（产品与背景有前后距离），纯平面设计稿或极浅景深实拍图效果可能不明显。

| 选项 | 含义 |
|---|---|
| 无虚化 | 前后景保持同等清晰度，适合需要完整展示场景细节的镜头 |
| 轻度虚化 | 背景轻微模糊，产品仍是视觉焦点，环境元素仍可辨认 |
| 强虚化 | 背景大幅模糊，产品被最大化突出 |

### 安全边距 —— 字段定义

控制运镜推进/拉远时，画面边缘与首帧图产品边缘之间预留的缓冲空间。数值越大，推进时产品边缘越不容易被推出画面外，但运镜的实际推进幅度观感会相应减弱。

- **0%**：不预留缓冲，按最大幅度执行，若产品边缘本来就贴近画面边界，风险最高
- **数值越高**：缓冲越多，运镜更保守，但更安全
- **建议默认8%**，具体应结合每张首帧图中产品实际占画面比例调整，产品占比越满，建议手动调高

## 五、S04 动态展示 —— 主方案/备用方案设计

### 设计背景

S04若采用真实翻页/展开的连续动作，需要首尾帧插值等更复杂的技术路径，与本方案单帧驱动的稳定性优先原则冲突。当前采用的折中方案是：**用一张定格在动作中间状态的首帧图（如"手正翻到一半"），配合极轻微的镜头运动，制造画面的动态错觉**，而非真实还原动作过程。这个方案的实际效果需要通过测试验证。

### 备用方案

若主方案（翻页定格）测试效果不理想，切换为备用方案：**产品与常见参照物（如手、桌面常见物件）同框展示**，用于让观众直观感受产品实际尺寸，这是电商产品视频中价值感较实在的常见手法。

### 切换机制

**主方案/备用方案的切换是模板级别的，不是单条产品链级别**——同一模板下所有批量克隆出的产品链，S04必须统一使用同一版本，不允许单条产品链单独覆盖，以保证批量场景下的风格一致性。测试阶段可以在模板配置界面对比两个版本效果，确定后固定下来。

### 对素材需求清单的影响

若最终采用备用方案，需要在素材需求清单中为S04新增一条收货要求：产品与参照物同框拍摄/制作的素材，这是主方案不需要的额外素材项，需要同步告知拍摄/制作团队。

## 六、属性面板交互设计

### 6.1 面板结构

每个分镜节点的属性面板分两种模式，通过标签页切换，默认展示常规模式：

**常规模式**：六个结构化字段（景别、运镜方式、光线氛围、运动幅度、背景虚化程度、安全边距），全部为下拉选择，不开放自由文本输入。安全边距字段仅在运镜方式为"推进"或"拉远"时显示。

**高级模式**：
- 可编辑提示词文本框：默认展示由常规模式六个字段自动组装而成的标准提示词，用户可以直接编辑修改
- 系统安全后缀（可选）：一个勾选框，勾选后会在最终提交生成的提示词末尾自动追加一句固定的安全约束描述（"保持产品结构与首帧内容一致，不脑补画面外结构"），默认勾选，用户可取消勾选用于A/B测试对比效果

### 6.2 常规模式与高级模式的联动规则

- 用户在常规模式调整任意字段，高级模式展示的提示词会自动重新组装
- 一旦用户**手动编辑**了高级模式的提示词文本，该节点进入"已自定义"状态：
  - 常规模式的六个字段变为禁用（避免用户误以为调整滑块仍然生效）
  - 节点卡片上显示"已自定义"角标
  - 提供"重置为标准模板"按钮，点击后清空自定义内容，恢复由结构化字段驱动
- 系统安全后缀的勾选状态独立于"是否自定义"，即使节点处于已自定义状态，安全后缀勾选框仍然可以单独控制

### 6.3 批量一致性保护

- 节点处于"已自定义"状态时，批量生成/合成前的预检面板中会汇总列出所有已自定义节点，与角色不匹配、时长超限等警告并列展示，要求用户显式确认后才能继续提交
- 测试阶段（安全后缀A/B对比），若同一批次内存在部分节点勾选、部分未勾选的情况，预检面板中应体现这一差异，便于测试结果与配置对应，避免事后无法追溯

### 6.4 模板保存策略

- "存为模板"时，默认**不保存**自定义提示词（`custom_prompt_override`），只保存结构化字段（六个常规字段）及其对应的标准组装规则
- 理由：自定义提示词通常是针对某个具体产品的个案调整，不适合作为可复用模板的默认内容，避免个案调整污染其他产品的批量生产
- 若后续发现"自定义提示词也需要模板级复用"的场景较多，可增加一个"是否连同自定义提示词一起保存"的勾选项作为可选行为，非默认

## 七、数据模型影响（供开发参考）

### 8.5 对素材需求清单的影响

两条产品线的S03、S04（含备用方案）、S05均需要各自独立的素材收货要求，不能共用同一套拍摄/制作清单。具体到S04：若某条产品线最终确认使用备用方案，只需为该产品线补充对应的"同框参照物"素材收货要求，不影响另一条产品线的收货计划（例如台历确认用备用方案不代表挂历也要切换）。

## 九、待测试/待确认事项

- [ ] S04主方案（翻页定格瞬间）的实际生成效果需要测试验证，确定是否可用，或需要切换到备用方案（尺寸参考同框），两条产品线需分别测试
- [ ] 挂历S04从原"全景摇镜"调整为"轻微平移+定格瞬间"后，是否仍能有效传达"整挂"的完整悬挂效果，需要专项测试确认，效果不足时考虑挂历专属的备用方案设计
- [ ] 系统安全后缀对生成结果稳定性的实际影响需要通过A/B测试验证（同一镜头分别在勾选/不勾选状态下对比）
- [ ] S02/S03细节镜头当前默认光线氛围改为"清透高饱和"，与原"侧光质感"的材质强调诉求存在方向差异，需要测试验证效果是否达到预期，不理想则考虑为细节镜头单独设置默认值
- [ ] 若任一产品线确认采用S04备用方案，需要更新对应产品线的素材需求清单，新增"产品与参照物同框"的收货要求
